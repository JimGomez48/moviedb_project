
MovieDB
==========


DESCRIPTION
------------
A simple movie database application using Python and the Django web framework.
Uses a MySQL backend.


AUTHOR
----------
James Gomez


REQUIREMENTS
-------------
### Python
Python version 2.7

### Database
MySQL v5.5

### Python Packages
* Django v1.8
* MySQL-python
* sqlparse
* django-datetime-widget

### Browser
any html5-enabled browser (I hope)
